# Bug Descriptions

## bug1.py
- **Intended Behavior**: Calculate the average of a list of numbers.
- **Current Issue**: ZeroDivisionError if the list is empty and incorrect sum initialization.

## bug2.js
- **Intended Behavior**: Fetch user data from an API and log the username.
- **Current Issue**: Asynchronous trap; trying to access data before the Promise resolves.

## bug3.cpp
- **Intended Behavior**: Reverse a string in-place.
- **Current Issue**: Out-of-bounds memory access due to incorrect loop boundary.

## bug4.py
- **Intended Behavior**: Check if a word is a palindrome.
- **Current Issue**: Logical error; it doesn't handle case sensitivity or spaces, and the string slicing is slightly off.
