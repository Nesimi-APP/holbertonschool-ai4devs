1. Mühit Məlumatları
IDE: Visual Studio Code (VS Code)

Əməliyyat Sistemi: Windows 10/11 / macOS / Linux

Alət: GitHub Copilot

2. İlkin Şərtlər
Aktiv GitHub hesabı.

GitHub Copilot abunəliyi (və ya fərdi şəxslər üçün ödənişsiz sınaq müddəti/tələbə paketi).

3. Quraşdırılma Addımları
Addım 1: IDE Genişlənməsinin (Extension) Yazılması
VS Code-u açın.

Sol paneldəki Extensions ikonuna klikləyin (və ya Ctrl+Shift+X).

Axtarış hissəsinə "GitHub Copilot" yazın.

Rəsmi GitHub tərəfindən təqdim olunan genişlənməni tapın və Install düyməsini sıxın.

Addım 2: GitHub Hesabı ilə Giriş
Quraşdırılma bitdikdən sonra aşağı sağ küncdə bildiriş görünəcək.

"Sign in to GitHub" düyməsinə klikləyin.

Brauzerdə açılan pəncərədə giriş edərək VS Code-a icazə verin.

Addım 3: GitHub Copilot Chat (Opsional)
Daha interaktiv təcrübə üçün GitHub Copilot Chat genişlənməsini də eyni qayda ilə quraşdırın. Bu, kod barədə suallar verməyə və izahlar almağa imkan verir.

4. Konfiqurasiya və Versiyalar
VS Code Versiyası: 1.85.0 və ya daha yeni.

GitHub Copilot Versiyası: v1.133.0 (və ya ən son stabil versiya).

Ayarlar:

Editor: Inline Suggest - Aktivləşdirildi (Enable).

İstəyə bağlı olaraq, müəyyən proqramlaşdırma dilləri üçün Copilot-u deaktiv etmək olar.

5. Yoxlama
Quraşdırılmanın uğurlu olduğunu yoxlamaq üçün yeni bir .js və ya .py faylı yaradın və şərh (comment) yazaraq Copilot-un təklif verib-vermədiyini müşahidə edin:

Python
# Create a function to calculate the area of a circle
Qeyd: GitHub Copilot-un ən yaxşı nəticə verməsi üçün kod yazarkən fayl adlarının və şərhlərin təsviri olması tövsiyə edilir.
