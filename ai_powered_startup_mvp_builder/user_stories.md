# User Stories - AI MVP Builder

1. **As a founder**, I want to describe my startup idea in plain text so that the AI can generate technical requirements for me.
2. **As a developer**, I want to select my preferred programming language (Python/JS/C++) so that the generated boilerplate matches my stack.
3. **As a product manager**, I want to generate a feature roadmap so that I can communicate the vision to stakeholders.
4. **As a non-technical user**, I want a simplified README file generated so that I can understand how the project works.
5. **As a user**, I want to edit the generated requirements before exporting so that they perfectly align with my needs.
6. **As a developer**, I want the AI to suggest potential API endpoints so that I don't miss core functional logic.
7. **As a security-conscious user**, I want a basic risk assessment report so that I can identify early vulnerabilities.
8. **As a busy entrepreneur**, I want to export everything directly to GitHub so that I can start coding immediately.
