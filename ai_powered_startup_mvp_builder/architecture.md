# System Architecture

The AI-Powered Startup MVP Builder follows a standard **Three-Tier Architecture**:

### 1. Presentation Layer (Frontend)
- React.js web interface for project configuration and visualization.

### 2. Logic Layer (Backend)
- FastAPI server handling AI orchestration (OpenAI/Gemini) and GitHub API integration.

### 3. Data Layer (Storage)
- PostgreSQL for persistent data and Redis for response caching.
